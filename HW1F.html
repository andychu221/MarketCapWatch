Standard Operating Procedure (SOP)

1. Define Bond Parameters (Pricing Tab)

  1. Go to the "Pricing" tab and select the "Bond" sub-tab.
  2. Enter the bond's basic information: Principal, Coupon Rate (%), Maturity (Years), and Coupon Frequency.

2. Define Call Schedule (Pricing Tab)

  1. Select the "Call Schedule" sub-tab.
  2. Enter the call schedule in text format (e.g., "NC1Yx1Y") in the "Call Schedule (Text)" field.
  3. The "Call Structure (Years)" field will auto-populate. You can also enter this manually.
  4. Enter the "Call Price(s)". If you enter a single value, it will be used for all call dates.
  5. For a Vanilla Bond: Clear the "Call Schedule (Text)" and "Call Price(s)" fields.

3. Input Market Data (Pricing Tab)

  1. Go to the "Discount Curve" panel on the right.
  2. Input the terms (Years) and zero-coupon rates (%) for the risk-free curve. This curve is used for discounting and tree calibration.
  3. (Optional) Input a Z-Spread in bps. This spread will be added to the entire curve to calculate the "Model Price".
  4. Press "Confirm" after making changes to the curve table.
  5. Go to the "Bond" sub-tab and enter the actual "Market Price" of the bond.

4. Set HW Model Parameters (Calibration Tab)

  1. Go to the "Calibration" tab.
  2. Set the "Mean Reversion (a)", "Volatility (σ)", and "Time Steps (per year)". These parameters are used globally for pricing.
  3. (Optional) Use the "Caplet Market Data" table to find optimal parameters. Input market data (Expire, Tenor, Strike, Black Vol).
  4. (Optional) Press "Run Optimization". The model will search for the optimal 'a' and 'σ' that best fit this market data. The parameters in the fields above will be updated.

5. Price the Bond & Analyze (Pricing Tab)

  1. Go back to the "Pricing" tab's "Bond" sub-tab.
  2. The model automatically calculates:
    • Model Price: Based on the Z-Spread (OAS=0).
    • OAS: The spread required to match the "Market Price" (using a base tree with Z-Spread=0 for solving).

6. Review Results

  1. Check the "Result" sub-tab to see the price breakdown (Vanilla, Option, Callable) based on the Model Price, along with the Effective Duration and the calculated OAS.
  2. Go to the "Tree" tab to visualize the interest rate tree, which is calibrated using the Z-Spread.

1. 定義債券參數 (Pricing Tab)

  1. 前往 "Pricing" 標籤頁，選擇 "Bond" 子分頁。
  2. 輸入債券的基本資訊：Principal (本金), Coupon Rate (%, 票面利率), Maturity (Years, 到期年限), 和 Coupon Frequency (付息頻V)。

2. 定義贖回條款 (Pricing Tab)

  1. 選擇 "Call Schedule" 子分頁。
  2. 在 "Call Schedule (Text)" 欄位中輸入文本格式的贖回結構（例如 "NC1Yx1Y"）。
  3. "Call Structure (Years)" 欄位將會自動填入。您也可以手動輸入。
  4. 輸入 "Call Price(s)" (贖回價格)。如果只輸入一個值，它將應用於所有贖回日期。
  5. 若為一般債券 (Vanilla Bond): 請清空 "Call Schedule (Text)" 和 "Call Price(s)" 欄位。

3. 輸入市場數據 (Pricing Tab)

  1. 在右側的 "Discount Curve" (折現曲線) 面板中。
  2. 輸入無風險利率曲線的期間 (Years) 和零息利率 (Rate %)。此曲線將用於折現和校準利率樹。
  3. (可選) 輸入 Z-Spread (bps)。此利差將被加到整個曲線上用來計算「模型價格 (Model Price)」。
  4. 修改曲線表格後請按下 "Confirm" (確認)。
  5. 回到 "Bond" 子分頁，輸入債券的實際「市場價格 (Market Price)」。

4. 設定HW模型參數 (Calibration Tab)

  1. 前往 "Calibration" 標籤頁。
  2. 設定 "Mean Reversion (a)", "Volatility (σ)" 和 "Time Steps (per year)"。這些參數將共用於所有定價計算。
  3. (可選) 使用 "Caplet Market Data" 表格來輔助尋找最佳參數。輸入市場數據 (Expire, Tenor, Strike, Black Vol)。
  4. (可選) 按下 "Run Optimization" (執行優化)。模型將搜索最適合此市場數據的 'a' 和 'σ'，並自動更新上方的參數欄位。

5. 債券定價與分析 (Pricing Tab)

  1. 回到 "Pricing" 標籤頁的 "Bond" 子分頁。
  2. 模型會自動計算：
    • Model Price (模型價格): 基於 Z-Spread (OAS=0) 計算。
    • OAS (選擇權調整利差): 根據 "Market Price" 反算出的利差 (反解時使用 Z-Spread=0 的基礎樹)。

6. 檢視結果

  1. 查看 "Result" 子分頁，可看到基於 Model Price 的價格分解 (Vanilla, Option, Callable)，以及有效存續期間 (Effective Duration) 和反算出的 OAS。
  2. 前往 "Tree" 標籤頁，可視化利率樹。此利率樹是使用 Z-Spread 校準的。
